# Maria Clar – Dein Wegbegleiter

Website für Maria Clar, zertifizierte Lebens- und Sozialberaterin.

## Technik

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- IONOS Deploy Now (Apache, SPA-Routing via public/.htaccess)

## Lokale Entwicklung

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
